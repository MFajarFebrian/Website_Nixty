## About the Project

This is an e-commerce platform for selling Office 365 subscriptions with features including:
- Product browsing
- User accounts
- Shopping cart
- Admin dashboard
- Secure checkout

Built with Nuxt.js and MySQL.

## Development

To update the site, make changes in the master branch, build the static files, and then update this branch.
